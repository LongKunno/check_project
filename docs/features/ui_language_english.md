# UI Language Standardization (English)

## Summary
All user-facing strings in the AI Static Analysis Dashboard have been migrated from Vietnamese to English, ensuring a consistent, professional, language-agnostic interface.

## Scope of Changes

| File | Type | Changes |
|------|------|---------|
| `Sidebar.jsx` | Layout | Nav labels, section headers, tooltips |
| `App.jsx` | Root | Button labels, page descriptions, loading states |
| `AuditView.jsx` | View | Error messages, section headers, KPI labels, empty states |
| `ProjectScoresView.jsx` | View | Page title, table headers, KPI strip, status pills, empty states |
| `MemberScoresView.jsx` | View | Page title, table headers, KPI strip, modal labels, empty states |
| `HistoryView.jsx` | View | Page title, table headers, action buttons, empty states |
| `SettingsView.jsx` | View | Page title, danger zone description, button labels, confirmations |
| `RulesConfigurator.jsx` | Component | Tab labels, tooltips, placeholders, toast messages, section headers |
| `useAuditJob.js` | Hook | Error messages, status messages |
| `chartHelpers.js` | Utils | Radar chart labels, bar chart labels |
| `TerminalLogs.jsx` | UI | Status messages, line count label |

## Translation Decisions

- **Retained identifiers unchanged** (e.g., `AI Sandbox`, `Scan All`) — already English.
- **Regex patterns in TerminalLogs** (e.g., `vi phạm`) kept unchanged — used to match backend log text, not displayed to users.
- **Internal code comments** not translated — irrelevant to user experience.
- **`getRatingColor` logic** (checking `Xuất sắc`, `Tốt`, `Khá`) — kept unchanged as these are backend-generated values. A follow-up backend change may be needed if rating labels are also to be Englified.

## Known Remaining Backend Dependency

The rating system (`Xuất sắc`, `Tốt`, `Khá`, `Trung bình`, `Yếu`) is generated by the Python backend in `scoring_engine`. These are matched in `getRatingColor()` functions. To fully English-ify the dashboard, the backend would need to return `Excellent`, `Good`, `Fair`, etc.

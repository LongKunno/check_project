# Interactive Dashboard (Giao diện trực quan)

Cung cấp cái nhìn 360 độ về chất lượng project thông qua dữ liệu thời gian thực và biểu đồ.

## Cách hoạt động
1. **Visualization**: Sử dụng **Chart.js** cho các sơ đồ Radar, Doughnut, và Line.
2. **Real-time API**: Giao tiếp trực tiếp với FastAPI backend để lấy kết quả phân tích.
3. **Violation Ledger**: Hiển thị chi tiết lỗi kèm snippet mã nguồn.
4. **Git Integration**: Hỗ trợ quét các repository từ GitHub thông qua API.

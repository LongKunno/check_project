🤖 AI Integration

AI Gatekeeper & Rule Engine

Three-Stage AI Pipeline

01
Stage 1 — Static Check

Regex + AST Scan

Quét siêu nhanh bằng Pattern Matching và Cây cú pháp Python. Phát hiện Hardcoded Secrets, Circular Imports, Code Smells.

02
Stage 2 — AI Gatekeeper

False Positive Filter

AI rà soát kết quả Stage 1. Nếu code context cho thấy an toàn → xóa bỏ cảnh báo giả. Giảm nhiễu triệt để.

03
Stage 3 — Deep Audit

Two-Pass Cross Verification

AI dò lỗi logic sâu. Nếu cần context cross-file → cắm cờ nghi vấn → Engine tìm source code hàm được gọi → AI xác nhận lần 2 (Pass 2).

📊 Scoring System

Hierarchical Scoring Mechanism

4 Quality Pillars — Thang điểm 10

🔒

Security

Hổng bảo mật, Secrets

K = 0.5
⚡

Performance

Tối ưu thực thi

K = 2.0
🛡️

Reliability

Xử lý lỗi, Ổn định

K = 2.0
🧹

Maintainability

Clean code, Structure

K = 4.0
Pillar Score = 10 / (1 + penalty / K_FACTOR)

Penalty được chuẩn hóa theo Effective LOC/1000 (Laplace Smoothing)

Cơ chế K-Factor & Tổng hợp

Trụ cộtK-FactorÝ nghĩa

  • Weighted Average by LOC — Module 10K dòng ảnh hưởng gấp 100× module 100 dòng.
